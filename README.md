# 数智平台与云农谷审批融合、数据同步

## 项目简介

这是一个展示数智平台与云农谷审批融合、数据同步的原型演示系统。

## 功能特性

- ✅ 数智平台（PC端管理系统）
- ✅ 云农谷APP（移动端应用）
- ✅ 企业微信（移动审批）
- ✅ 小程序（轻量级应用）

直接打开 `index.html` 文件即可在浏览器中查看。

- `index.html` - 主页面文件
- `styles.css` - 样式文件
- `script.js` - 脚本文件
